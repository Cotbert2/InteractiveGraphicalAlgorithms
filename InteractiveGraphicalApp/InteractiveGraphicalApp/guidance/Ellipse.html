Midpoint Ellipse Drawing Algorithm

Efficient rasterization of ellipses using integer arithmetic and symmetry

📘 What is the Midpoint Ellipse Algorithm?

The Midpoint Ellipse Drawing Algorithm is an efficient method to draw ellipses using integer operations only. It is a variation of the midpoint circle algorithm adapted for ellipses. The algorithm divides the first quadrant of the ellipse into two regions and calculates pixel positions based on decision parameters, exploiting the ellipse's symmetry.

📐 Formulas

Let the ellipse have radii a (horizontal) and b (vertical), centered at (0, 0).

Region 1 (dy/dx < 1):
Initial decision parameter:
p1 = b² - a²b + 0.25a²

For each step:
If p1 < 0:
  x = x + 1
  p1 = p1 + 2b²x + b²
Else:
  x = x + 1
  y = y - 1
  p1 = p1 + 2b²x - 2a²y + b²

Region 2 (dy/dx ≥ 1):
Initial decision parameter:
p2 = b²(x + 0.5)² + a²(y - 1)² - a²b²

If p2 > 0:
  y = y - 1
  p2 = p2 - 2a²y + a²
Else:
  x = x + 1
  y = y - 1
  p2 = p2 + 2b²x - 2a²y + a²

The algorithm plots all four quadrants using the symmetry of the ellipse: (±x, ±y). This reduces computation and ensures accurate rendering.
