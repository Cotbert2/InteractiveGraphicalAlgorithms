Flood Fill using Breadth-First Search (BFS) fills a region by expanding in all directions from a starting point. It uses a queue to process all neighboring pixels evenly, layer by layer.

Queue<Point> queue = new Queue<>();
queue.Enqueue(startPoint);
while (queue.Count > 0) {
  Point p = queue.Dequeue();
  // Check bounds and color
  queue.Enqueue(neighbors...);
}

üìê Concept

BFS fills the area radially using a queue.
- Visit the seed pixel and check color.
- Add all 4 neighbors to the queue.
- Process each pixel layer by layer.
