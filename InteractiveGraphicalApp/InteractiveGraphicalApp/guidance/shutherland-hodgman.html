Sutherland–Hodgman Polygon Clipping Algorithm

Efficient Polygon Clipping Against a Convex Window

📘 What is the Sutherland–Hodgman Algorithm?

The Sutherland–Hodgman algorithm is used for clipping polygons against a rectangular or convex clipping window. It works by successively clipping the polygon edges against each boundary of the clipping window.

📐 Clipping Strategy

For each edge of the polygon:
- If both vertices are inside → output the second vertex
- If going from inside to outside → output the intersection point
- If going from outside to inside → output the intersection point and the second vertex
- If both are outside → output nothing

The process is repeated for all four boundaries (left, right, top, bottom) of the clipping window.

🧠 Summary

  • Best suited for convex clipping windows.
  • Outputs a new list of vertices forming the clipped polygon.
  • Maintains polygon integrity if input and clip window are convex.
