Cohen–Sutherland Line Clipping Algorithm

Efficient Line Clipping in a Rectangular Window

📘 What is the Cohen–Sutherland Algorithm?

The Cohen–Sutherland algorithm is used in computer graphics to clip lines to a rectangular viewport. It uses region codes to efficiently determine whether a line is fully inside, outside, or intersects the clipping window.

📐 Region Codes and Logic

Each point has a 4-bit code:
Top = 1000, Bottom = 0100, Right = 0010, Left = 0001

If code1 | code2 == 0 → Line is completely inside
If code1 & code2 != 0 → Line is completely outside
Otherwise → Line needs to be clipped

The algorithm repeatedly clips the line segment until it either becomes trivially accepted or rejected.
